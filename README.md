Blinkit Sales Dashboard (Power BI)
📊 Overview

This project presents an interactive Power BI dashboard built to analyze sales performance for Blinkit outlets. It provides insights into total sales, average sales, item distribution, ratings, and outlet-level performance across multiple dimensions such as outlet size, location tier, and item types.

🎯 Objectives

Monitor overall sales performance

Compare outlet types and sizes

Analyze product category contribution

Evaluate customer ratings and item distribution

Identify trends in outlet establishment over time
